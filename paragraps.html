Paragraph tag

The (p) HTML element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent locks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.

Html Formating Elements

HTMl Formating Elements are used to make text design in HTML like Bold text (b), Italic text (i), Marked text (mark), Small text (small), Delete text (del), Insert text (ins) etc.

Paragraph and Formating Elements Explanation

p

 The (p) tag defines a paragraph. Browsers automatically add a single blank line before and after each (p) element.

br

 The (br) HTML element produces a line break in text, (br) can break a line of sentence inside or out side the paragraph (p).

b

 The Bold text (b) HTML Formating Element make a certain text Bold, usually used to emphasize something strongly in writing.

i

 The Italic text (i) HTML Formating Element make a certain text Italic. Italics are used for emphasis or contrast, to draw attention to some particular part of a text.

mark

 The Marked text (mark) HTML Formating Element make a certain text Marked or highlighted for reference or notation purposes, due to the marked passage's relevance or importance in the enclosing context.

small

 The Small text (small) HTML Formating Element used to set Small font size. It decreases the font size by one size.

del

 The Delete text (del) HTML Formating Element make a certain text Delete. it is a text that has been deleted from a document. Browsers will usually strike a line through deleted text.

ins

 The Insert text (ins) HTML Formating Element make a certain text Insert. The Insert tag defines a text that has been inserted into a document. Browsers will usually underline inserted text.
