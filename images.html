Images

in order for webpages to be more exppresive, programmers could put images on their webpages.

Tag/Attributes Explanation
Img Tag The "< img>" tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image.
src attribute The src attributes specifies the path to the image. It could either be Relative path or Absolute. it coud also be a link to an img on the wab.
alt Attribute alt attribute is used to describe what the picture is incase the image didn't load properly. it also helps with things like page reader.
width Attribute Width attribute is used to specify teh widhth of an image. Although CSS could also modify the width of an image.
height Attribute The height attribute is used to specify the height of an image. Although CSS could also modify the height of an image.
