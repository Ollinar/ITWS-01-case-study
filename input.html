Input Types

Input tag is what really makes up the Html form. It is the part where users actually interact and input their informations. There are tons of input types out there and this will only cover some of the most used ones.

List of common input types

Input type Explanation
Text This is the most basic input type where the user can type anything; Be it letters, numbers, or special characters.
Number This is an input field specialized for nubmers. User would also be given inside the input box an arrow up and down to increment or decremanet the input number.
Date This is an input field specialized for dates. User would also be given a calendar to help them choose a date.
Checkboxes This is an input type where users dont actualy type anyting and just click the checkbox or label. Checkboxes could be grouped together by putting the same name attribute on each one.
Radio Buttons Just like checkboxes, however radio buttons can only have one selected item and must be grouped together by having the same name attribute.
Email This input is just like text type but has some features that helps user fill it out with email. Like in mobile, it will bring up a different keyboard that's more specialized in filling out email address.
Password This input is just like text type but whatever the user type will be hidden from the screen with dots.
Telephone number This input field is specialized in taking contact numbers and the programmer could specify how the number should be formatted using the pattern attribute.
Search This input type is like the text type but with features that specialize on search. This will change the action key in mobile.
URL This input type is just like the text type but with features specializing on URLs like checking if it's valid and such.
Select & Option/Datalist This input type has its own tag "< select> and < option> or < datalist>". This tag will create a dropdown menu that users can select from.
Color This input type will give the user a color pallet to choose from.
Range This input type will give the user an slider to move.
File This input type is where the user can upload their files such as images and documents.
Submit This input type is like a button. When pressed will submit the form.
Reset This input type is like a button. When pressed will clear all of the input fields.
