Headings

HTML header tags are used for the headings and for design (h1,h2,h3,h4,h5,h6) of a page from the rest of the content. These tags are also known to webmasters as heading tags or simply header tags. Headers differentiate in size whitch "(h1)" is the biggest and "(h6)"is the smallest. header can change color, placement and font depending on the style the users want.

Type of Header Explanation

h1

 This is the largest header, often used for title in html.

h2

 This is the second largest, often used for secondary title in html.

h3

 This header is smaller to the h2 header, often use for definition title or list title.

h4

 This header is third to the smallest header.

h5

 This header is hardly used because of it's size.

h6

 This header is the smallest header in html, almost same as a paragraph (p) comand size.
